NeuralChat

A lightweight AI chatbot built using HTML, CSS, and JavaScript, powered by Groq APIs. The goal of this project is to create a clean, fast, and interactive chat experience similar to modern AI assistants.


---

What it does

Streams responses in real time

Supports multiple chat sessions

Saves conversations in browser storage

Allows switching between different models

Lets users enter their own API key

Includes stop generation functionality



---

How to run

1. Download or clone the project


2. Open index.html in your browser


3. Enter your Groq API key


4. Start chatting



You can get an API key from: https://console.groq.com


---

Tech used

HTML

CSS

JavaScript

Groq API (OpenAI-compatible)



---

Notes

No backend is used

Everything runs in the browser

API key is stored locally (localStorage)
